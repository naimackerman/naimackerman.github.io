# Personal Website

The production website is the Vite app at the repository root. The `Personal Website/`
directory is the original design export and should not be opened directly in a browser;
its local JSX loading will be blocked by browser CORS rules.

## Local development

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, normally `http://localhost:3000/`.

## Production preview

```bash
npm run build
npm run preview
```
